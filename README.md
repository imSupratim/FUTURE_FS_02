# MINI CRM WEBSITE

A **Mini Customer Relationship Management (CRM)** Web Application built using the **MERN Stack** that helps businesses manage leads, contacts, and customer interactions efficiently.  

The system allows users to register, login, track leads, update customer information, and manage sales pipelines through a clean and responsive interface.

---
## 🚀 Features

### 🔐 Authentication System
- User Registration  
- Secure Login with JWT  
- Protected Routes  

### 👥 Lead Management
- Add new leads  
- View all leads  
- Update lead status  
- Delete leads
  
### 📊 Dashboard
- Overview of CRM data  
- Quick access to leads and contacts  

### 🧑‍💼 Customer Management
- Store customer details  
- Track communication history  

### 🎨 Modern UI
- Responsive design  
- Smooth navigation using React Router  

---

## 🛠️ Tech Stack

**Frontend**
- React.js  
- React Router  
- Axios  
- CSS / Tailwind  

**Backend**
- Node.js  
- Express.js  

**Database**
- MongoDB  
- Mongoose  

**Authentication**
- JWT (JSON Web Token)
